# 🤖 Brazo Robótico Controlado por Joystick

![Demo del proyecto](fotos/demo.gif)

## 📋 Descripción
Este proyecto implementa un brazo robótico de 2 grados de libertad controlado mediante un joystick analógico. Los movimientos del joystick se traducen en posiciones para dos servomotores, permitiendo un control intuitivo y en tiempo real.

## 🎯 Características
- Control de 2 servomotores mediante joystick
- Mapeo suave de movimientos (sin saltos bruscos)
- Indicador LED de estado
- Fácil de construir y programar
- Base expandible para más servos

## 🛠️ Materiales Necesarios
| Componente | Cantidad | Observaciones |
|------------|----------|---------------|
| Arduino Uno | 1 | Cualquier modelo compatible |
| Servomotor SG90 | 2 | Micro servos estándar |
| Módulo Joystick | 1 | KY-023 o similar |
| LED | 1 | Color a elección |
| Resistencia 220Ω | 1 | Para el LED |
| Protoboard | 1 | Pequeña/mediana |
| Cables jumper | 20 | Macho-macho y macho-hembra |
| Fuente 5V | 1 | Para alimentar servos |
